## 术语解释

+ **selenium IDE**，firefox 下的一款插件，用于录制 test case， 用于回放自动测试
+ **Selenium WebDriver**，即 Selenium 2，使用专用 driver 直接调取浏览器 API 来进行测试
+ **Selenium RC**， 即 Selenium 1 （是一个http代理程序）。因为使用 JavaScript 来进行测试，因此需要本地起一个http代理来避免跨域问题
